## Setup Instructions

1. Clone the repository
2. Install dependencies:
   ```bash
   bundle install
   ```
3. Set up the database:
   ```bash
   rails db:create db:migrate
   ```
4. Start the server:
   ```bash
   rails server
   ```
5. Visit `http://localhost:3000` in your browser
